# Oblivia

Chef de mémoire officiel de Néron.

## Architecture

Néron
↓
Oblivia
├── SQLite (mémoire nerveuse)
└── Obsidian (mémoire autobiographique)

## Rôles

SQLite :
- runtime
- agents
- goals
- événements
- historique

Obsidian :
- identité
- décisions
- roadmap
- journal
- leçons apprises

## Objectif

Fournir un point d'accès unique à la mémoire de Néron.

## Évolution future

- MCP Memory Server
- recherche sémantique
- résumés automatiques
- mémoire long terme
